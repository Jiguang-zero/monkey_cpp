# 用`c++`重构`用Go语言自制解释器`

## 前言
本项目仅供学习交流使用。

关于用C++重构这本书 《用Go语言自制解释器》这本书，已经通过邮件取得作者 ` Thorsten Ball `的同意。应作者要求，在这里附上这本书的链接。  

这本书的链接如下。 [Writing An Interpreter In Go](https://interpreterbook.com)

## 编译
本项目是一个 CMake 项目，C++ 标准为 C17，具体项目配置详看 `CMAKELists.txt` 文件。  
项目的启动文件为 `main.cpp`。
测试函数为 `test.h` 中定义的相关函数。  

## 工具
本项目使用 Clion 进行开发。
